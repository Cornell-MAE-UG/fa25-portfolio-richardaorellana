Problem 3 from problem set 3 focused on the use of thermal resistance networks to analyze different insulation conditions on a metal alloy. This was the first assignment that asked explicitly for this method to be used. I believe this method to be incredibly powerful for elementary analysis of linearized thermal systems in steady state, which is how it has been used in subsequent problem sets. The parallels between electrical systems and thermal systems became so obvious whiel I was doing this problem.
